# 동성특수인쇄 인스타그램 디자인 스펙 (감도 규칙서 v1)

> 레퍼런스·트렌드 리서치로 뽑은 "감도 규칙". 렌더러(`render-ig.mjs`)와 힉스필드
> 이미지 프롬프트가 **이 규칙을 따른다.** 반응도 데이터로 매 사이클 갱신(선순환).
> 출처: 2026 캐러셀/에디토리얼 트렌드(seamless·hook·15~20단어) + 한국 감성 카드뉴스
> 관례(여백·2컬러·큰 타이포·저장하고 싶은 완성도) + 레퍼런스 계정 특징.

## 0. 문제 진단 (왜 지금 감도가 아쉬운가)
- 스타일마다 **컬러가 제각각**(네이비/레드/그린) → 피드가 안 묶임. 프리미엄은 **절제된 통일 팔레트**.
- **글자·이모지 과다** → 여백 부족, 프리미엄 톤 저하. 이모지는 카드에서 최소화(캡션으로).
- 폰트가 산세리프 단일 → **에디토리얼 감**이 약함. 명조(세리프) 디스플레이로 럭셔리 감 추가.
- 커버가 "후킹 공식"이 약함. 0.5초 안에 스크롤을 멈추는 **볼드 훅 + 큐리오시티 갭** 필요.

## 1. 통일 브랜드 팔레트 (전 카드 공통 · 3~5색만)
| 역할 | 값 | 용도 |
|---|---|---|
| Ink (딥네이비) | `#0c1526` | 기본 배경·본문 앵커 |
| Ivory (아이보리) | `#f4efe6` | 밝은 캔버스·본문 텍스트(딥 배경 위) |
| Gold (골드) | `#c9a86a` | 강조 1순위(작게) — 키워드·라인·라벨 |
| Signal (레드) | `#e2503f` | 강조 2순위(아주 작게) — 포인트 1개만 |
| Mist (뮤트 그레이블루) | `#9fb0c6` | 서브텍스트·캡션 |
- **원칙**: 한 카드에 강조색은 **1개만**(골드 or 레드), 나머지는 무채. 채도 높은 색으로 큰 텍스트 금지.
- 사진 위 스크림/그라데이션도 이 팔레트 안에서(딥네이비 계열)로 통일 → **피드 일관성**.

## 2. 타이포그래피 (세리프 디스플레이 + 산세리프 본문)
- **디스플레이(제목)**: `Nanum Myeongjo`(명조, 700/800) — 럭셔리 에디토리얼 감. 큰 스케일 + 좁은 행간(1.15).
- **본문·라벨·핸들**: `Pretendard`(500~800). 라벨은 **작게 + 자간 넓게(letter-spacing .2em, 대문자 영문)**.
- **위계는 색이 아니라 스케일·굵기·여백으로**. 제목 82~104px, 서브 30~36px, 라벨 26~30px.
- 키워드 1~2개만 골드 강조(`*키워드*`). 제목에 이모지 금지(캡션에서만).

## 3. 레이아웃 · 여백 (여백이 프리미엄)
- 캔버스 1080×1350(4:5). 좌우 세이프 84~96px, 상단 110~120, **하단 세이프존 ≥135**(크로미움 잘림 방지).
- **슬라이드당 한 가지 메시지**. 텍스트는 **한국어 12단어 이내**(제목+서브 합쳐). 빽빽함 금지.
- 넓은 네거티브 스페이스로 "시각적 쉼" → 세련도↑. 사진은 클로즈업/와이드를 섞어 리듬.
- **동일 그리드**(같은 여백·같은 라벨 위치·같은 핸들 위치)로 전 슬라이드 통일 → 하나의 브랜드 화면.

## 4. 커버(1장) 후킹 공식
`볼드 훅 헤드라인(한국어 6~10자, 최대 텍스트)` + `패턴 인터럽트(고대비 사진/골드 포인트)` + `큐리오시티 갭(다 안 보여주기)`.
- "나를 위한 콘텐츠인가?"와 "넘기면 뭘 얻나?"에 0.5초 안에 답. 이모지·해시태그 없음.
- 예: "열었을 때, *브랜드*가 완성됩니다" / "같은 로고가 *3배* 비싸 보이는 법".

## 5. 사진(힉스필드) 감도 규칙
- **에디토리얼 럭셔리 제품컷**: 톤 통일(따뜻한 딥네이비·아이보리·골드 라이팅), 얕은 심도, 소프트 그림자, 매크로 디테일.
- **글자 금지**(`no text no letters`) — 글자는 항상 HTML 오버레이.
- 컷 종류를 섞기: 매크로(질감/후가공) + 미들(제품 단독) + 무드(언박싱/손). 같은 색보정으로 **피드 일관성**.
- 재사용 금지 — 글/슬라이드마다 신규.
- 모델: 프리미엄 제품컷은 상위 모델(§ 이미지 모델 스펙) 사용, 스타일 프리셋 통일.

## 6. 3스타일 = "레이아웃"만 다르게, 팔레트·톤은 공통
- **Editorial(구 cinema)**: 전면 사진 + 하단 명조 대형 제목 + 골드 헤어라인. 무드/언박싱.
- **Catalog(구 photoA)**: 상단 얇은 골드 룰 + 라벨(영문 대문자 자간) + 명조 제목. 제품 단독컷.
- **Feature(구 photoB)**: 사진 상단 + 하단 아이보리(밝은) 밴드 + 딥네이비 명조 제목. 매크로 디테일.
- 셋 다 **같은 팔레트/톤/핸들 위치** → 스타일은 다르되 한 브랜드로 읽힘.

## 7. 마감 장표 · 캡션
- 마감 장표(`ig-outro.mjs`)는 이 팔레트로 유지(딥네이비+골드+레드 1포인트).
- 캡션: 30대 여성 실장 톤 + 이모지(카드 아님 캡션에서), AI 생성 고지, 해시태그 10개 내외.

## 7-1. 이미지 모델 · 프롬프트 템플릿 (아트디렉션 = 감도의 최대 레버)
- **모델**: `soul_2`, **quality 2k**(1080p→2k로 상향), aspect `3:4`(4:5 카드에 맞게 크롭), count 1.
  - `marketing_studio_image`는 실제 제품사진 레퍼런스가 필요해 제외. 정책(403) 열리면 상위 모델
    (seedream/nano_banana) 실사비교 후 교체 검토.
- **통일 아트디렉션(모든 컷 공통 접미어)** — 피드 일관성의 핵심:
  `editorial product photography, warm deep-navy and ivory palette, soft gold rim light,
   shallow depth of field, soft diffused studio light, subtle film grain, premium catalog quality,
   minimal clean composition, no text no letters`
- **컷 타입별 앞부분(주제)만 교체**:
  - 매크로: `extreme macro of <후가공/질감>, tactile detail`
  - 제품 단독: `single <제품> centered on <재질> surface, generous negative space`
  - 무드: `hands <행동> a <제품> in moody light, cinematic still`
- **금지**: 글자/로고 텍스트, 과도한 채도, 잡다한 소품. **하나의 피사체 + 넓은 여백**.
